# 智能SQLer v1.0.0 发布说明

## 项目概述

智能SQLer是一个强大的AI辅助SQL工具，能够帮助用户通过自然语言生成SQL语句，优化现有SQL查询，解释复杂SQL的含义，并提供丰富的模板系统来提高工作效率。

## 核心功能

### 1. AI驱动的SQL生成
- 通过自然语言描述自动生成SQL语句
- 支持多种数据库类型：MySQL, PostgreSQL, SQL Server等
- 可根据表结构信息生成更精确的SQL

### 2. SQL优化
- 分析现有SQL查询并提供性能优化建议
- 识别潜在的性能瓶颈和优化空间
- 生成优化后的SQL语句

### 3. SQL解释
- 解释复杂SQL语句的功能和执行逻辑
- 提供通俗易懂的中文解释
- 帮助用户学习SQL语法和最佳实践

### 4. 模板系统
- 预定义常用SQL查询模板
- 支持自定义和管理用户模板
- 基于模板快速生成SQL

### 5. 安全防护
- 实施严格的SQL注入防护机制
- 输入验证和长度限制
- 敏感操作监控

## 技术栈

### 后端
- **语言**: Rust
- **Web框架**: Actix-Web
- **数据库连接**: 支持多种数据库
- **AI集成**: OpenAI GPT API

### 前端
- **框架**: Svelte
- **语言**: TypeScript
- **样式**: Tailwind CSS

## 部署指南

### 环境要求
- Rust 1.70+
- Node.js 16+
- npm 8+
- 支持的数据库系统

### 后端部署
1. 配置环境变量或修改配置文件
2. 构建应用：`cargo build --release`
3. 运行服务：`cargo run --release`

### 前端部署
1. 安装依赖：`npm install`
2. 构建应用：`npm run build`
3. 部署到静态文件服务器

## API文档

详细的API文档可以在部署后的`/api/docs`路径下访问。
